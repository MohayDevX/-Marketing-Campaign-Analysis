# 📊 Customer Campaign Analysis

## 📌 Project Overview
Analyzed 2,240 customers using Python to identify profitable segments and evaluate marketing campaign performance for ROI optimization.

## 🔍 Key Insights
| Factor | Finding |
| --- | --- |
| **Ideal Customer** | University graduates, avg age 54.5 |
| **Best Campaign** | Response campaign: 15.03% success rate |
| **Worst Campaign** | AcceptedCmp2: 1.35% success rate |
| **Core Correlation** | Higher income strongly correlates with higher total spending |

## 💡 Business Recommendation
Immediately stop low-performing campaigns and reallocate 70% of ad budget to target graduates aged 50-55. Projected to 3x ROI.

## 🛠️ Tools Used
- **Python** - Core programming language
- **Pandas** - Data cleaning and analysis
- **Matplotlib** - Data visualization
- **Jupyter Notebook** - Development environment

## 📈 Dataset
Marketing Campaign Dataset - 2,240 customers with demographics, spending, and campaign responses.
